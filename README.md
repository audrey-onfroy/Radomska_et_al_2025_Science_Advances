### Glial-to-mesenchymal transition of tumor Schwann cells drives the genetic burden in MPNSTs from Neurofibromatosis type 1 mouse model

This repository contains the code to ensure the traceability of the analysis of single-cell RNA Sequencing data from the raw count matrices to the figures in the associated publication. Please open the [index.html](https://audrey-onfroy.github.io/Radomska_et_al_2025_Science_Advances/) file in any browser, and navigate through all files in this repository.

The results behind this analysis are published in Science Avances:

> Radomska *et al.*, Glial-to-mesenchymal transition of tumor Schwann cells drives the genetic burden in MPNSTs from Neurofibromatosis type 1 mouse model, 2025, *Science Advances*; doi: [10.1126/sciadv.adt9210](https://doi.org/10.1126/sciadv.adt9210)
